At a Glance

A structured instructional model that translates decades of cognitive science into practical classroom strategies for teachers in low-resource settings across Africa and the Global South. No internet. No devices. No specialist software — just intentional teaching.

The Five Principles

Each principle is designed to work within existing timetable structures — no extra workload beyond intentional planning at the start of term.

SSpaced Retrieval
Come back to it.

Revisit prior learning across deliberately planned time intervals. The space between sessions is where memory consolidates.

PProgressive Reinforcement
Build it up, layer by layer.

Each retrieval session strengthens and extends prior learning. The second session is harder than the first — learning compounds.

AActive Recall
Pull it from memory — don't peek.

Students reconstruct knowledge from memory rather than re-reading notes. That effortful retrieval is what strengthens the memory trace.

CCognitive Consolidation
Give the brain time to settle.

Space new content so earlier material has time to solidify. Reduces cognitive overload and builds durable understanding.

EEvidence-Based Adjustment
Look at what's working. Change what isn't.

Track retrieval scores across the term and adjust spacing or task design when improvement stalls. A notebook is all you need.

Getting Started: Three Steps

You don't need new resources or extra time. These three steps integrate spaced practice into what you already do.

1
Map your retrieval points

At the start of term, mark four weeks for a 10-minute retrieval task. Weeks 1, 4, 8, and 12 of a 13-week term are a strong default — the expanding intervals match how memory consolidates.

2
Design tasks that require recall

Students must produce answers from memory — not recognise them. Paper quizzes, oral questioning, and think-pair-share all work without computers or internet access.

3
Record scores and adjust

Note class average scores at each retrieval point. If scores at Week 8 are not higher than Week 4, adjust your spacing intervals or task design for the following term.

Research Basis

The framework synthesises three bodies of cognitive science evidence into principles any classroom teacher can apply directly.

Spacing Effect

Distributing practice across time produces stronger long-term retention than massed practice. The optimal gap scales with how long you want the learning to last (Cepeda et al., 2006).

Cognitive Load Theory

Spacing reduces cognitive load over time as students build schemas incrementally, freeing working memory for deeper understanding of new content (Sweller et al., 1998).

Retrieval Practice

Active recall — not re-reading — is the mechanism that makes spacing work. Effortful retrieval strengthens memory traces far more than passive review (Roediger & Butler, 2011).
